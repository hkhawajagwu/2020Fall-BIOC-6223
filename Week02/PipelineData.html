Sample Virus Pipeline

The following is a condensed example that might be taken from a methods section.

Note that a "dict" file will have a mediatype of "application/octet-stream" (the default value). Others are easy to find.



This pipeline to identify baseline single nucleotide polymorphisms (SNPs), (insertions), and (deletions) that correlate with reduced antiviral drug efficacy in (Hepatitis C virus subtype 1) works by first aligning viral reads to a reference genome. Choice of reference genome is critical to identifying potential drug-resistant mutations. Once small read sequences are mapped, a variant map is produced by identifying SNPs. The SNP map can be compared between individuals with or without drug resistance to ledipasvir to identify possible causal mutations."


All data is stored on a HIVE instance on a linux server. Starting with read file 1 and reference file 2, align the input file 1 to input file 2 using the Hexagon tool, using a minimum match length of 75, and allowing for 15 percent mismatches. This will produce several files. Take the alignment file and use Heptagon to call SNPs, setting "collapseRpts" to "false," "cutEnds" to "do not cut," and "supportedDeletions" to "true." Heptagon produces three files, all_indels.dict, FreqProfile.csv, and SNPprofile. All of the files are appended to the base URL https://hive2.biochemistry.gwu.edu/dna.cgi?cmd=objFile&ids=14782&filename=all_indels.dict.
